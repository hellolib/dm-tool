# dm-tool
达梦数据库工具（数据库管理工具、数据库迁移工具）

## 快速使用
1. clone 项目
2. 进入tool目录
3. 运行相关命令
```bash
bash dts # 数据库迁移工具
bash manager  # 数据库管理工具
```

## 使用docker镜像运行
1. 拉取镜像
```bash
docker pull hub.deepin.com/wuhan_udcp/dm-tool:v1.0.0
```
2. 运行相关命令
```bash
bash run.sh dts # 数据库迁移工具
bash run.sh manager  # 数据库管理工具
```
